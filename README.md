# 💫 About Me:
JavaScript/TypeScript 기반으로 Express, NestJS, PostgreSQL을 사용해 안정적인 백엔드 서비스를 구축해왔어요. API 설계, 데이터 모델링, Docker·Nginx 기반의 운영 환경 구성 등 서비스 개발 전반을 경험했고, 시스템 구조를 효율적으로 설계하는 데 관심이 많아요.

최근에는 여러 서비스에서 공통으로 사용할 MSA 형태의 중앙 결제 서버를 설계·구축하며 API_KEY 관리, 보안 설계, 결제 흐름 설계 등을 맡았어요. 또한 별도로 RabbitMQ 기반의 드론 실시간 데이터 처리 파이프라인을 Node.js와 FastAPI로 구현해 실시간 처리 구조를 직접 구축해봤어요.

새로운 기술을 빠르게 흡수하고 실제 서비스에 적용하는 데 강점이 있어요.
